# Rich Result by Job Id

The Rich Result by Job Id component allows you to display the result of a Runnable by providing its [Job](../../core_concepts/20_jobs/index.mdx) id.

![Rich Result by Job Id](../../assets/apps/4_app_component_library/rich_result_by_job_id.png "Rich Result by Job Id")

The following section details Rich Result by Job Id component's specific settings. For more details on the App Editor, check the [dedicated documentation](../0_app_editor/index.mdx) or the App Editor [Quickstart](../../getting_started/7_apps_quickstart/index.mdx):

## Rich Result by Job Id configuration

| Name        | Type | Connectable | Templatable | Default | Description                 |
| ----------- | :--: | :---------: | :---------: | :-----: | --------------------------- |
| Job Id |  string  |    true     |      false       |         | The [job id](../../core_concepts/20_jobs/index.mdx) of the run to display results. |

## Outputs

| Name    |  Type   | Description                         |
| ------- | :-----: | ----------------------------------- |
| result  |     string    | The result of the runnable.   |
| loading | boolean | The loading state of the component. |
| Job Id |  string  | The [job id](../../core_concepts/20_jobs/index.mdx) of the run to display results. |
